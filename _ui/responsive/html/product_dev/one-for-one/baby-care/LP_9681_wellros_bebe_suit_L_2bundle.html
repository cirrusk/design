FAQ
  1. 1) ‘생산자 직접 배달’ 이란 무엇인가요?
    • 생산자 직접 배달이란, 제품 생산자가 한국암웨이 부산 물류센터로 제품을 입고한 후 소비자에게 일괄적으로 출고되는 일반 제품들과는 다르게 생산자가 주문 데이터를 전달받아 제품을 직접 배달하는 서비스를 말합니다.
  2. 2) 웰로스 베베 수트 번들은 6만원 미만인데도 배달 수수료가 면제되나요?
    • 생산자 직접 배달 서비스 품목은 수수료 면제 품목으로, ABO가격이 6만원 미만일지라도 배달 수수료가 면제됩니다.
  3. 3) 웰로스 베베 수트 번들과 생산자 직접 배달 품목이 아닌 제품을 동시에 구매해도 배달 시점이 다를 수 있나요?
    • 웰로스 베베 수트 번들은 생산자 직접 배달 품목으로 유한킴벌리에서 출고되고, 생산자 직접 배달 품목이 아닌 다른 제품들은 한국암웨이 부산 물류센터에서 출고되어 배달되기 때문에 도착 시점이 상이할 수 있습니다.
