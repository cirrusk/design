ABO 자녀정보등록

향 후 진행될 2세대 자녀를 대상으로 하는 교육 및 이벤트를 위해서 자녀의 정보를 정확하게 입력해 주시기 바랍니다.

  • Step1 1. 정보 수집에 대한 본인 확인 및 수집 활용 동의서 양식 다운로드 양식 다운로드
  • Step2 2. 정보 수집에 대한 본인 확인 및 정보 수집 활용 동의서 출력 및 작성
  • Step3 3. 정보 수집 내용 등록하기 버튼 클릭 후 해당 항목 입력(다이아몬드 또는 자녀가 직접 입력)
  • Step4 4. 정보 수집에 대한 본인 확인 및 정보 수집 활용 동의서 촬영 및 업로드(정보 수집 관련 내용 등록 페이지에서 업로드)
  • Step5 5. 내용 최종 확인 후 페이지 내의 완료 버튼 클릭
  • Step6 6. 내용 수정 및 삭제 시 정보 수집 내용 등록하기 페이지 내의 내용 변경 후 해당 버튼 클릭
