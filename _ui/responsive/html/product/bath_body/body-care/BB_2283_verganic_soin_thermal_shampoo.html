베르가닉 스왕 떼르말 샴푸

유기농 식물 성분인 캐모마일꽃수와 알로에베라즙을 함유해 피부 진정효과를 전해주는 샴푸입니다.
풍부한 미네랄을 함유한 프랑스 몽브룅뢰벵 온천수와 칼렌듈라 꽃 추출물 등 매우 순한 성분들을 엄선하여 만들었으며, 연약한 아기와 어린이 헤어를 부드럽게 씻어내 깨끗하고 윤이 나도록 케어 해줍니다.

  • 스왕(SOIN) : 케어(CARE)
  • 떼르말(THERMAL) : 온천을 이용한(SPA)
베르가닉 스왕 떼르말에 함유된
프랑스 프로방스 지역의 몽브룅뢰벵 온천수

몽브룅뢰벵 온천수는 로마 시대부터 스파로 유명한 프랑스 프로방스 지역의 온천 휴양지로 몽방투(Mount Ventous:방투산) 끝자락에 위치하고 있습니다.
몽브룅뢰벵의 온천수(EAU THERMALE MONTBRUN®)는 프랑스 국가에서 엄격한 기준을 가지고 직접 관리하고 있으며, 풍부한 미네랄이 함유되어 있다고 알려져 있습니다.

추천 대상자
  1. 연약하고 순수한 아기와 어린이 피부
주의사항
  1. 1) 화장품 사용 시 또는 사용 후 직사광선에 의하여 사용부위가 붉은 반점, 부어오름 또는 가려움증 등의 이상 증상이나 부작용이 있는 경우 전문의 등과 상담할 것
  2. 2) 상처가 있는 부위 등에는 사용을 자제할 것
  3. 3) 눈에 들어갔을 때에는 즉시 씻어낼 것
  4. 4) 사용 후 물로 씻어내지 않으면 탈모 또는 탈색의 원인이 될 수 있으므로 주의할 것
보관방법
  1. 1) 어린이의 손이 닿지 않는 곳에 보관할 것
  2. 2) 직사광선을 피해서 보관할 것
