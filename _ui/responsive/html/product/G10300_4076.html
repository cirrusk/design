사용순서
STEP 1 스포이드 뚜껑을 열고 함께 동봉되어 있는 비타민 C 캡으로 교체해서 끼워줍니다.
STEP 2 비타민 C 캡을 닫은 후, 상단을 눌러 비타민 C 파우더가 액체에 들어갈 수 있도록 해줍니다.
STEP 3 순수 비타민 C 파우더가 액체에 잘 녹을 수 있도록 30초간 잘 흔들어줍니다.
STEP 4 비타민 C 캡을 제거하고 스포이드 뚜껑을 다시 끼워 줍니다.
STEP 5 아침/저녁으로 내용물을 충분히 2스포이드 취하여 손바닥에 떨어 뜨린 후 얼굴에 골고루 펴바릅니다.
