간이과세자의 부가가치세 계산사례

세무법률 목록

총괄등록자의 경우 한국암웨이(주)는 부가가치세를 신고할 때 실제의 매출액을 파악하기 어려워 ABO가 구입한 금액(부가가치세 포함)에 3%를 더한 금액을 과세표준으로 신고하고 있습니다.

예) 직전년도 12개월 동안 회사로부터 매입한 상품가액 : 13,200,000원 (부가가치세 1,200,000원 포함)

구분 금액
공급대가 { 회원가격(부가세포함) + 3%(소매마진) } 13,596,000원
(b) 납부세액 (공급대가의 1%) 135,960원
(c) 매입세액 공제 (매입세액의 10%) 120,000원
차감 세액 (b-c) 15,960원
