ABO의 유형별 부가가치세 납세의무

1. 사업자형 ABO(부가가치세법상 사업자)

  • - A type ABO : 다단계 판매업에 대하여 사업자 등록을 하고, 인적 또는 물적 시설을 갖추고 다단계 판매 등의 사업을 영위하는 자
  • - B type ABO : 다단계 판매업 이외의 사업(일반 자영업 등)에 대하여 사업자 등록을 하고 다단계 판매업이외의 사업을 영위하는 자
  • 1) 다단계판매업자(Amway)에게 회원 등록 시 또는 등록 후 소매업을 영위할 목적으로 다단계판매업자에게 소매사업자로 신고한 자
    (납세관리인 선정 신고서 작성)
  • 2) 개별사업장을 별도로 가지고 있는 자
  • 3) 다단계판매업자에게 도매업 또는 도,소매 겸업자로 신고한 자

2. 자가소비형 ABO(소비자)

  • - C type ABO : 사업자 등록을 하지 않고 다단계판매업자(한국암웨이)로부터 구입한 상품을 직접 소비하고 소매업을 영위하지 않는 자 (부가가치세 신고 · 납부 의무가 없음)
