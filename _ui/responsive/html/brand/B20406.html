경제적인 유지비

불필요한 전기요금까지 줄여주는 경제적인 유지비

평상시엔 대기전력을 최소화하고 정수할 때만 효율적으로 전기를 쓰는 인스턴트 ON 시스템 개발
전기 낭비를 막아 유지비가 부담없는 이스프링만의 좋은 물 기술입니다.

(전기요금 기준 : 2014년 6월)

전기요금 기준 : 전기요금, 초기 구매 비용, 렌탈료, 5년간 총 비용, 1L당 평균 비용, 1일 평균 비용 정보제공
전기요금* 하루 1시간 작동시(소비전력 60W, 대기 전력 2W)
한달 전기요금 ₩640 X 12개월 = ₩7,680
1년 전기요금 ₩7,680 X 5년 = ₩38,400
초기 구매 비용 ₩1,050,000 (이스프링 퍼싯타입 기준)
렌탈료 없음
5년간 총 비용 5년간 4회 필터 교체
₩258,000 X 4개 = ₩1,032,000
1L당 평균 비용 ₩2,120,400
1일 평균 비용 정보제 ₩2,120,400 / (5,000L X 5년) = ₩84.82
1일 평균 비용 정보제 ₩2,120,400 / (365 X 5년) = ₩1,162

* 한달 전기요금은 이스프링 정수기 사용에 따라 추가되는 금액이며 1개월 기본 전기 사용량 273KWh (지식경제부 2013sus 1dnjf 21일 발표 ‘2011년 가구당 월평균 전기사용량’)을 기준으로 산출하였습니다.
(한전 사이버지점 홈페이지 참조) 사용량에 따라 차등 적용되는 누진세로 인하여 실제 청구되는 전기요금과 차이가 있을 수 있습니다.

** 1년에 5,000리터 사용 기준
