총 (18)건
비밀번호는 어떻게 찾을 수 있나요?
  1. ① 아이디 찾기와 동일하게 본인확인 후 비밀번호 찾기를 진행할 수 있으며 본인확인이 완료되면 새로운 비밀번호로 변경할 수 있습니다.

본인확인 과정은 아이디 찾기와 동일합니다.

답변이 충분하지 않으셨다면 이메일 문의하기를 이용해 주세요.

보너스 연관 반품은 어떻게 확인 하나요?

[보너스 연관 반품]

  1. ① 조회 월 선택 : 보너스 연관 반품 조회 월을 선택할 수 있습니다. (이전월은 기본적으로 선택됨)
  2. ② 상세보기 : 반품상세정보 확인을 원하는 회원의 반품 상세정보를 확인할 수 있습니다.

답변이 충분하지 않으셨다면 이메일 문의하기를 이용해 주세요.

제품 상세화면 안내
  1. ① 바로구매 : 주문 수량 선택 후, 장바구니에 담지 않고 바로 결제하여 제품을 구매합니다.
  2. ② 장바구니 : 주문 수량 선택 후, 제품을 장바구니에 담습니다.
  3. ③ 위시리스트 : 관심 상품을 위시리스트에 담아 관리합니다.
  4. ④ SNS 공유 기능 : 카카오톡, 카카오 스토리, 밴드, 페이스북으로 제품 정보를 공유합니다.

답변이 충분하지 않으셨다면 이메일 문의하기를 이용해 주세요.
